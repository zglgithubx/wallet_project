## 工程简介
业务背景：电商业务中，需要给电商app设计一个用户钱包，用户可以往钱包中充值，购买商品时用户可以使用钱包中的钱消费，商品申请退款成功后钱会退回钱包中，用户也可以申请提现把钱提到银行卡中。
  
用程序实现如下api接口:
* 查询用户钱包余额
* 用户消费100元的接口
* 用户退款20元接口
* 查询用户钱包金额变动明细的接口
## 项目介绍
### 技术栈  

SpringBoot+Mybatis+Swagger
### 启动步骤
* 修改数据库连接的账户和密码
* 直接启动（启动项目前会自动建库建表）
* 进入Swagger接口文档测试接口
